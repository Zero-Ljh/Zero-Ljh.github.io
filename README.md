# Junhui Li — Personal Homepage

李军辉 个人主页 · Wuhan University of Science & Technology

Built with pure HTML/CSS/JS, zero dependencies. Deployed on GitHub Pages.

## Quick Start

```bash
# Just open index.html in a browser
open index.html
```

No build step, no dependencies, no configuration.

## Structure

```
├── index.html      # Main page
├── css/style.css   # All styles
├── js/
│   ├── data.js     # Content data (edit this to update content)
│   ├── i18n.js     # Bilingual system
│   ├── router.js   # Hash routing for sub-views
│   └── main.js     # Init, navigation, animations
└── assets/         # Images, resume PDF
```

## Update Content

Edit `js/data.js` — no need to touch HTML or CSS for content changes.
